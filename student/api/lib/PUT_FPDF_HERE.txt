IMPORTANT:
You need to download the FPDF library to generate PDF results.

1. Go to: http://www.fpdf.org/en/download.php
2. Download the latest ZIP file (v1.86).
3. Extract the ZIP file.
4. Copy the 'fpdf.php' file and the 'font' folder.
5. Paste them DIRECTLY into this folder (api/lib/).

Your folder structure should look like:
/api
  /lib
    - fpdf.php
    - font/
    - PUT_FPDF_HERE.txt
